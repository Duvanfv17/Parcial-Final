Pasos para crear el proyecto en git hub usando la terminal de vs code:

Paso 1: inicializar git en el proyecto.
git init

Paso 2: Agregar todos los archivos
git add .

Paso 3: Crear un commit
git commit -m "Primer commit"

Paso 4: Conectar el repositorio con git hub
git remote add origin (Link del repositorio)

Paso 5: Subir el Proyecto:
git push -u origin main
